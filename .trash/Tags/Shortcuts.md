#shortcuts

## General shortcuts

> ctrl + a : select all
> ctrl + c : copy
> ctrl + v : paste
> ctrl + x : cut
> ctrl + z : undo
> mod + <arrow_keys> : move window


## Browser shortcuts
#browsers

> alt + <- : previous page
> alt + -> : next page
> ctrl + t : new tab
> ctrl + tab : switch to next tab
> ctrl + w : close tab
> ctrl + f : find in page

### Firefox shortcuts

>shift+ctrl+period : increase playback speed
>shift+ctrl+comma : decrease playback speed
>ctrl + k : search using browser
## Hyprland shortcuts
#hyprland #arch #linux 

>mod + q : open terminal
>mod + c : close tab
>mod  + e : open file manager
>mod + d : open wofi(app lookup)
>mod + j : switch tab orientation
>mod + shift + <num> : move tab to <num> workspace
> mod + v : free-float tab
