[[Networks & Web Programming]]

## physical layer

- data rate

- bit synchronization

- configuraion > simple point-to-point links, a multipoint configuration

- topology > a full mesh topology, a star topology, etc

- mode > simplex, half dupelex, full duplex
---
## data link layer

• Framing

• Physical addressing.

• Error detection, error correction.

• Access Control (MAC is treated as a sublayer with its MAC
addresses).

• Flow Control

---
## Network Layer

- Form packets (encapsulated with IP headers).

- Network addressing (unique IP addresses).

- Routing through different networks (Source-to-destination delivery).

- Fragmentation.

- Best effort delivery.
---
## Transport Layer

• Process addressing and multiplexing, also known as “service-point
addressing,”
• Segment handling, message is divided into segments, each segment has a
sequence number used to put the message back together at the
destination.

• Connection control, The transport layer can be connectionless or
connection-oriented.

• Flow control.

• Error control

---
## Application Layer

• Session support ( half/full duplex sessions, session history, connection
information set between end systems)

• Internal representation conversion (data formats for different systems)
