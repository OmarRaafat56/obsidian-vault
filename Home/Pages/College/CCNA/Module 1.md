# Module 1 – Networking Today

**Course:** Introduction to Networks (ITN v7.0)
**Module Objective:** Explain the advances in modern technologies and how networks impact people, businesses, and industries.

---

## 🌍 1.1 Networks Affect Our Lives

### Networks Connect Us

* Networks connect people globally like never before
* Communication is as critical as basic human needs
* Networks enable:

  * Global communities
  * Collaboration
  * A “human network” without boundaries

### No Boundaries

* Location no longer limits:

  * Communication
  * Education
  * Business
  * Innovation

---

## 🧩 1.2 Network Components

### Hosts (End Devices)

* Any device that sends or receives data
* Examples:

  * PCs, laptops
  * Smartphones, tablets
  * Servers, printers

### Server Roles

| Server Type  | Purpose                  |
| ------------ | ------------------------ |
| Email Server | Sends and receives email |
| Web Server   | Hosts websites           |
| File Server  | Stores and manages files |

* **Clients** request services from servers

---

### Peer-to-Peer (P2P) Networks

* Devices act as both client and server
* Used in **very small networks**

**Advantages**

* Easy to set up
* Low cost
* Less complex

**Disadvantages**

* No centralized control
* Not scalable
* Less secure
* Slower performance

---

### Intermediary Network Devices

Devices that connect end devices and manage data flow:

* Switches
* Routers
* Wireless access points
* Firewalls

**Functions**

* Regenerate signals
* Determine best path
* Detect and report errors

---

### Network Media

How data travels between devices:

| Media Type  | Description           |
| ----------- | --------------------- |
| Copper      | Electrical signals    |
| Fiber-optic | Pulses of light       |
| Wireless    | Electromagnetic waves |

---

## 🗺️ 1.3 Network Representations & Topologies

### Network Diagrams

* Use symbols to represent devices and connections
* Key terms:

  * **NIC** (Network Interface Card)
  * **Port**
  * **Interface** (often interchangeable with port)

### Topology Diagrams

* **Physical topology:** device locations & cabling
* **Logical topology:** IP addressing, ports, data flow

---

## 🌐 1.4 Common Types of Networks

### Networks by Size

* **Small Home Network**
* **SOHO** (Small Office/Home Office)
* **Medium/Large Enterprise**
* **World Wide Network** (Internet)

---

### LAN vs WAN

| LAN                   | WAN                      |
| --------------------- | ------------------------ |
| Small geographic area | Large geographic area    |
| High-speed            | Slower links             |
| Single owner          | Service provider-managed |
| Internal access       | Interconnects LANs       |

---

### Internet

* Global collection of interconnected LANs & WANs
* Uses copper, fiber, and wireless
* Not owned by anyone

**Organizations that help manage it**

* IETF
* ICANN
* IAB

---

### Intranets & Extranets

* **Intranet:** private internal network
* **Extranet:** controlled access for external partners

---

## 🛡️ 1.6 Reliable Networks

### Network Architecture

Technologies and protocols that move data across the network.

### Four Characteristics of a Reliable Network

1. **Fault Tolerance**

   * Multiple paths
   * Packet-switched networks
2. **Scalability**

   * Grows without performance loss
   * Uses standards & protocols
3. **Quality of Service (QoS)**

   * Prioritizes voice & video traffic
4. **Security**

   * Protects devices and data

---

---

### Cloud Computing

* Enabled by data centers

**Cloud Types**

* **Public**

* **Hybrid**
* **Custom (industry-specific)**

---

### Home Technology Trends

* Smart homes
* Powerline networking
* Wireless broadband (WISP, cellular-based)

## 🔐 1.8 Network Security

### Security Threats

**External**

* Malware
* Zero-day attacks
* DoS attacks
* Identity theft

**Internal**

* Lost devices
* Accidental misuse
* Malicious insiders

---

### Security Goals (CIA Triad)

* **Confidentiality**
* **Integrity**
* **Availability**

### Security Solutions

* Antivirus / antispyware
* Firewalls
* ACLs
* IPS
* VPNs
